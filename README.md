# NestJS EmbedJs example

This project is built as an example to show how to use [EmbedJs](https://github.com/llmembed/embedjs) in a NestJs application.

## Getting started

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
